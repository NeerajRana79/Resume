WhatsApp Message Scheduler

Python Twilio API Automation

üìã Project Overview

This project is a Python-based automation script designed to schedule and send WhatsApp messages at a specific date and time. It uses the Twilio API for reliable message delivery.

The main goal was to learn API integration, automated task scheduling, and secure credentials management in Python.

‚öôÔ∏è Key Features

  • Message Scheduling: Set exact dates and times for WhatsApp messages to be sent autonomously.
  • Twilio Integration: Reliable message delivery using the Twilio API.
  • Environment Variables: Securely manage API keys and credentials.
  • Beginner-Friendly: Clean, well-structured, and easy-to-understand code.

üíª Code Sample

Here's a snippet showing the core scheduling logic:

scheduler.py 
import os
from twilio.rest import Client
import datetime
import time

# Load credentials from environment
TWILIO_ACCOUNT_SID = os.getenv('TWILIO_ACCOUNT_SID')
TWILIO_AUTH_TOKEN = os.getenv('TWILIO_AUTH_TOKEN')
client = Client(TWILIO_ACCOUNT_SID, TWILIO_AUTH_TOKEN)

def send_whatsapp_message(to_number, text):
    message = client.messages.create(
        body=text,
        from_='whatsapp:+14155238886',
        to=f'whatsapp:{to_number}'
    )
    print(f"Message sent! ID: {message.sid}")

def schedule_message(target_time, to_number, text):
    while True:
        current_time = datetime.datetime.now()
        if current_time >= target_time:
            send_whatsapp_message(to_number, text)
            break
        time.sleep(60) # check every minute
